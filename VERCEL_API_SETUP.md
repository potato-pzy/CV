1. Open your project on Vercel
2. Go to **Settings** → **Environment Variables**
3. Add a new variable:
   - **Name:** `VITE_WEB3FORMS_ACCESS_KEY`
   - **Value:** your Web3Forms access key (3062a272-a6d3-4041-8521-7cd48342648f)
   - **Environment:** Production  
4. Click **Save**
5. **Redeploy** the project (Deployments → ⋮ → Redeploy) so the new variable is applied
